# Architecture Decision Records

Architecture Decision Records (ADRs) capture durable technical decisions that shape MIGI’s safety, provenance, data, and deployment model.

## Status Values

```text
proposed
accepted
superseded
deprecated
```

## Format

Each ADR should contain:

1. Context
2. Decision
3. Consequences
4. Alternatives considered
5. Migration or review notes
